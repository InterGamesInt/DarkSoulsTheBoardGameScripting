# Dark Souls: The Board Game – автоматизация для Tabletop Simulator

Этот проект добавляет полную автоматизацию настольной игры **Dark Souls: The Board Game** в **Tabletop Simulator**.  
Скрипты самостоятельно подготавливают стол: выкладывают тайлы локаций, раскладывают карты стычек, настраивают персонажей и управляют боссами в зависимости от выбранного режима.

## Особенности

- **Два режима игры**  
  - *Стандартный*: игроки сражаются с мини-боссом, затем с главным боссом.  
  - *Кампания*: пошаговое прохождение сценариев из официальных дополнений (более 20 кампаний).

- **Выбор наборов тайлов**  
  Можно активировать/деактивировать:  
  * Core Tiles 
  * Darkwood Tiles 
  * Iron Keep Tiles 

- **Выбор наборов карт стычек**  
  Поддерживаются:  
  * Core Encounters  
  * Darkroot Encounters  
  * Iron Keep Encounters  
  * Explorers Encounters  

- **Автоматическая расстановка**  
  Тайлы вытягиваются из сумок случайным образом и стыкуются дверными точками.  
  На каждый тайл кладётся карта стычки соответствующего уровня.

- **Выбор персонажа**  
  После подготовки стола каждому игроку предлагается выбрать класс (Knight, Warrior, Assassin и т.д.).  
  Скрипт автоматически:
  * переключает планшет игрока на нужное состояние,
  * выдаёт стартовое снаряжение (оружие, броню, щиты, аксессуары),
  * раскладывает жетоны эстуса, удачи, героических действий,
  * размещает фигурку на тайле костра.

- **Поддержка мега-боссов**  
  Для боссов типа *Stray Demon*, *Black Dragon Kalameet*, *Four Kings* и других корректно настраивается поле боя.

## Установка

1. Скачайте файлы репозитория и поместите их в папку `Saves` вашей Tabletop Simulator.
2. Убедитесь, что на столе присутствуют все необходимые объекты.
3. Загрузите игру и нажмите на кнопку `⚙️ Настройки`.

## Использование

1. В открывшемся окне выберите режим:
   - **Стандартный** – укажите мини-босса и главного босса, выберите наборы тайлов и карт стычек.
   - **Кампания** – выберите один из сценариев.
2. Установите количество игроков (от 1 до 4).
3. Нажмите **Применить и начать**.
4. Скрипт:
   - расставит тайлы и карты стычек,
   - выставит счётчик искр (в зависимости от числа игроков),
   - предложит каждому игроку выбрать персонажа.
5. После выбора персонажей фигурки появятся на костре – можно начинать игру.
## Структура файлов
- **Global.lua**  
  Содержит:
  * данные о боссах (`bossData`) и сценариях кампаний (`campaignScenarios`),
  * функции для работы с тайлами, дверными точками, картами стычек,
  * UI выбора персонажа,
  * настройку планшетов игроков,
  * вспомогательные утилиты.

- **Settings.lua**  
  Отвечает за:
  * создание кнопки открытия меню,
  * генерацию XML интерфейса с актуальными списками боссов и сценариев,
  * обработку событий (выбор режима, боссов, тогглов),
  * вызов глобальных функций `setupStandardGame` или `setupCampaignGame`.

- **SettingsUI.html**  
  Шаблон окна настроек. Используется скриптом для динамического формирования UI.
